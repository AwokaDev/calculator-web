# Simple Calculator

A minimalist web-based calculator with theme switching support.
Built with pure JavaScript to practice **DOM manipulation** and **CSS Variables**.

## Features

- **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
- **Theme Switcher:** Toggle between Light and Dark modes instantly.
- **Error Handling:** Gracefully handles errors (like division by zero).
- **Responsive Design:** Clean UI that works on different screens.

## Demo
Try it live here: [Link to your GitHub Page](https://awokadev.github.io/calculator-app/)

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Flexbox layout & CSS Variables for theming.
- **JavaScript (ES6+):**
  - `document.querySelector` for DOM access.
  - `addEventListener` for user interactions.
  - `try...catch` block for safe calculations.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/AwokaDev/calculator-app.git
   ```
2. Open `index.html` in your browser.
