# Simple Calculator

A minimalist web-based calculator with theme switching support (Light/Dark mode). Built with vanilla HTML, CSS, and JavaScript.

## Features
- **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
- **Theme Switcher:** Toggle between Light and Dark modes.
- **Responsive Design:** Clean and simple UI.
- **Error Handling:** Gracefully handles errors (like division by zero or invalid input).

## Demo
Check out the live version [here](https://awokadev.github.io/calculator-web/)

## Technologies Used
- **HTML5:** Structure of the application.
- **CSS3:** Styling and Flexbox layout.
- **JavaScript (ES6+):** Logic and DOM manipulation.
